Cladogram Generator

Cladogram Generation is based on a custom-built Merge Sort with Common Ancestor Backtracing and Derived Characteristics Pruning Algorithm created in Python. Demo Link can be found here(https://replit.com/join/prkmzusdhq-24sdas). The idea of this algorithm started from the Nova Evolution Lab(https://www.pbs.org/wgbh/nova/labs/lab/evolution/). The Github Page can be accessed here(https://github.com/Saptak625/CladogramGeneratorAlgorithm).
